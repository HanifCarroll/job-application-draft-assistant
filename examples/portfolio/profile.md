# Example Freelancer

Product-minded full-stack engineer who helps teams turn ambiguous software projects into scoped, reliable releases.

Positioning:
- Builds production web apps, internal tools, and AI-assisted workflows.
- Works well with existing codebases, product constraints, and non-ideal legacy systems.
- Communicates tradeoffs clearly and keeps implementation tied to business risk.

Working style:
- Start by mapping the current system, user flow, risks, and backlog.
- Prefer small, reversible fixes before large rewrites.
- Keep evidence visible when using AI or automation.
